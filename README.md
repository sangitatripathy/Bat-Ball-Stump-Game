# 🏏 Bat-Ball-Stump Game

A fun browser-based mini game based on Rock-Paper-Scissors with a cricket twist — choose **Bat**, **Ball**, or **Stump** and try to beat the computer!

## 📚 How to Play

- Choose between:
  - 🏏 Bat
  - 🎾 Ball
  - 🪵 Stump
- Rules:
  - Bat beats Ball
  - Ball beats Stump
  - Stump beats Bat
- If both player and computer choose the same: It's a draw!
